[docker-image-transfer-helper](https://github.com/null-box/docker-image-transfer-helper) (DIT) is a CLI tool for transferring [Docker](https://www.docker.com/) images between container registries without requiring a local Docker daemon. It simplifies cross-registry image migration workflows in CI/CD pipelines and air-gapped environments.
