[git-write-mcp](https://github.com/ipolyzos/git-write-mcp) is a [Model Context Protocol](https://modelcontextprotocol.io/) stdio server written in Go. It exposes a curated, allowlisted set of Git write operations and optional GitHub REST endpoints to AI coding agents, enabling safe and controlled repository automation on Linux and macOS.
