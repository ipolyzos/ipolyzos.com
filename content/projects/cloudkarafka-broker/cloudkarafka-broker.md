[cloudkarafka-broker](https://github.com/ipolyzos/cloudkarafka-broker) offers hosted publish-subscribe messaging systems in the cloud as on-demand [Cloud Foundry](https://www.cloudfoundry.org/) service instances. The broker does not include an [Apache Kafka](https://kafka.apache.org/) server or cluster. Instead, it is meant to be deployed alongside a managed [CloudKarafka](https://www.cloudkarafka.com/) account, which provides provisioning and maintenance of [Apache Kafka](https://kafka.apache.org/) instances through APIs.
