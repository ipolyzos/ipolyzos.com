[azure-cloudshell-setup](https://github.com/ipolyzos/azure-cloudshell-setup) is a collection of shell scripts that automate the configuration of the [Azure Cloud Shell](https://docs.microsoft.com/en-us/azure/cloud-shell/overview) environment. It provisions the shell with the tools, aliases, and settings needed for Kubernetes and cloud-native development on Azure.
