text-search-mcp is a [Model Context Protocol](https://modelcontextprotocol.io/) stdio server written in TypeScript. It exposes read-only, schema-versioned text search over allowlisted filesystem roots via ripgrep (with grep fallback), giving AI agents structured search capabilities without unrestricted filesystem access.
