archive-mcp is a [Model Context Protocol](https://modelcontextprotocol.io/) stdio server written in Go. It wraps tar, unzip, and zipinfo in a safe, structured, read-only interface — allowing AI agents to inspect and extract archive contents without unrestricted shell access.
